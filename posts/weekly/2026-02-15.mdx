# 💻 Cladude Code 初尝试

在2025年的年初，我直接订阅了cursor的一年制，所以在过去的一年里我都是cursor的重度用户，然后在这周第一次尝试了Cladude Code。

我一直算是一个比较拥护GUI工具的人，包括以前使用Git的时候也是比较喜欢使用GUI工具，不用记复杂的命令也可以清晰直观的看到所有的操作以及状态，
我过去从Copilot到Cursor，就是因为我想要清晰的看到Diff，然后方便我的review以及及时的修改  

但是现在AI生成的代码质量也愈来越高，在彻底明白我的要求的时候，他生成的代码绝大部分时间都是不需要我去做Review的，甚至是需要我去学习的

刚好订阅的时间也快到了一年的时间，我从中转站买了200的额度用上了了Claude Code，然后让他帮我修复项目中所有的lint错误以及ts规范，跟Cursor不一样的是，Claude Code的一个任务从头到尾给我感觉是没有中断过，超长上下文😮，
相比之前Cursor经常一个对话跑着就开始Loading不动真是折磨我的耐心；在模型切换方面，得益于CC Switch的强大，也可以在做不同事的时候快速切换不同的模型  

总的来说，Claude Code Cli 的使用体验现在还是挺好（除了烧钱，4.6猛猛改了两天直接把我200大洋的额度干完了），或许在未来的某一天编程会变成一个黑盒的事情，到时候程序员亲手写代码何尝不是一个传统手工艺

# 🗺️ 故近生疏

我的童年是在一条公交线路上，从7号线的头几个站，沿途路过体育中心、华贸，接着过了桥，到了惠州西湖站，这个时候可以下车过步行街，躲过路边吆喝的销售后后到里头的新华书店在里面挑一本有意思的漫画或者书坐一下午，接着径直穿过步行街可以到另一边的水东街，那边的糖水小吃比较出名，但是对于还是学生的我有点囊中羞涩；
如果不进步行街，也可以拐进隔壁每年过年都有花灯，人满为患的惠州西湖；如果想要去更远一点，可以坐多几个站到滨江公园，这边有个不大不小的游乐场跟另一家较大的新华书店，书店的门口往往都有几个大爷摆摊卖糖画还有棉花糖，里面基本可以找到外面买不到的习题册跟辅导书，一度让我苦不堪言；

我记忆里我童年的周末大多就是在这么一条线路上，来回4块钱，嘴馋买个糖或者买个甜筒也可以在10块钱左右，会获得一个相对充实的下午。

但是现在已经是大学四年、工作三年后，已经很久没有留意附近的变化，趁着过年的机会自己一个人重新走了一趟，甚至已经感到陌生，滨江公园的游乐场拆了，过去写作业的咖啡厅也换了位置，步行街的新华书店也早已拆掉，华贸旁新建了个市民公园，很多东西在不知不觉中就消失了，平淡无常的某个午后或许就是最后一次的踏足，但也有一些新东西在悄然出现

# 🚶 香港行

在广东待了20多年，但是没有去过一次香港，一直都想过去一趟，倒也不是说想看看那边的景色，主要是想开一些港卡（汇丰、中银、众安，其实就是把能开的都开了😊）（2月17日补充：中银把我拒了😭，另外两个过了），顺势逛逛香港跟给家里人带点东西。跟我挺久没见的堂哥堂嫂还有他的朋友一块同行，
我们早上8点多从惠州出发，一路驱车到深圳皇岗口岸，接着转大巴进港

香港的人很多，但是地方是真的不大，并且一些地方非常的老旧，街边任意张贴的广告、表演海报还有各种喷漆涂鸦，在夜幕降临时，拥挤的高楼下逐渐亮起的彩色霓虹灯牌，让我感到恍惚想到以前电视剧里头光怪陆离的香港夜梦

![香港街头](/postImg/2026-02-15/DSC_2234.JPG)

香港的的士、公车都很有意思，除了他们本身饱和度很高的红色特别吸睛以外，很多铁轨上的电车以及公车都十分的mini，我的mini指的是他们宽度，他们宽度很窄，但是却是有两层的高度  

后面才知道这里边是有一些历史原因，一个是当年英国殖民时期从英国本土引进的标准，香港电车采用窄轨设计；另一方面就是香港高密度的建筑，城市快速发展，道路便难以大规模拓宽，综合下来就成现在这样了

![香港叮叮车](/postImg/2026-02-15/DSC_2224.JPG)

另一个有意思的社会现象还跟我们到的时间有关系，我们是周日到的香港，到了以后在中环以及一些隧洞里面可以看到很多菲律宾在打地铺以及摆摊，有的在开着手机直播拍抖音或者一块唱歌跳舞，如果是周一到周六的话还见不到这个现象  

这些人其实是菲律宾籍家庭佣工，香港从上世纪70年代开始引入外籍家庭佣工，然后他们的工作方式就是每周工作六天，从周一到周六，负责日常家务与照护工作，香港的法律是要求`同住`的，所以周一到周六的时间里他们是住在雇主的家里

周日是他们的休息日，加上菲律宾本身是一个社交性很强、群体感很强的文化，所以他们在周日的时候会摆摊聚在一块，组成“临时的家”。

这件事情在香港已经持续几十年，是一种非常稳定的社会现象。香港的双职工家庭可以全职投入工作，老人孩子依然可以有人照顾，是这些家庭佣工承担了大量“再生产劳动”，这些佣工早已经是香港中产阶级得以存在的支柱了吧。

随着年龄的增大，早已意识到乌托邦是不存在的，所有社会的稳定，都建立在某种分工之上。只是有些分工更可见，有些更不可见。
