**已经是第五年写年终总结，每次写的时候回想自己过去的一年总是思绪万千，现在在深夜，仅用文字记录自己过去的一年以及这一刻**

# 回顾去年立的flag
+ 依然是好好学习技术上的东西，学会某个框架 / 技术的同时，搞懂其背后的原理
+ 折腾一些 Side Project / 折腾自己的博客
+ 多看看书，多沉淀，将书里的内容转换为自己的东西
+ 理财，季度性的盘点自己的账单
+ 好好说话，谨慎言行。
+ 工作上有所突破
+ 培养一门特长
+ 学习英语
+ 好好锻炼
+ 身体健康

今年在技术上更细致了一些，会从性能指标以及数据上去进行分析后再针对性地做一些事情，了解了一些框架他们为什么要做这么一个事✍️，总体上来说思考更深了一点，但是还得练

虽然今年上没有特别多的锻炼，只有偶尔周末折腾下自己，但是好在一年下来身体没有什么毛病，身材体重也没有什么变化的样子🙈

书读的貌似没有去年那么多，除了自己开始学钢琴以外，今年看的书大部分理论比较多，所以也比较难啃（甚至有点囫囵吞枣的感觉

# 2024

> 我还会走许多弯路，还会为许多“已实现”感到失望。但一切终将实现它们的意义。

## 工作
今年在工作上是颇累的一年，业务负责的事情跟去年大差不差，在流程里边有一些自己的想法，试试明年能不能落地下来💻  

另外就是今年自己一些额外的技术栈（NestJS、Java、Go）都有在内部的一些项目上进行实践；在前端性能方面也自己做了一些尝试也取得了一些成果，并且做了轮子方便其他同事使用，这算是对自己比较满意的一些地方。

比较不满意自己在工作上情绪控制上多少还是有点问题，我的+1也跟我说过，**不要把情绪带入到工作当中**，明年要告诫自己也要控制自己👐。

今年也明显感觉到现在整体的大环境越来越高压，但危机也是机遇，在自己可控的范围内用压力push自己前进也是一种成长的方式。

> 熬不住的时候，才正是修行时

## Code
今年比往年更专注在了业务实现，还有内部监控/提效之类的角度上，所以在技术深度上没有什么提升，但是在业务实现以及监控优化上有了一些新的了解以及方向，也算是拓宽了自己的一些视野

拓宽自己的技术栈或者实现方法不是什么坏事，多会一点东西在遇事的时候就多一种选择

今年也是满满当当写了1300+h的代码🤯

## 读

今年最有感而发的是《**献给阿尔吉侬的花束**》。

> 虽然一些人因为他们接受的教育或者他们先天的智慧，在智商上或者一些领域上已经是专家，但在情感上还是小孩子，他们无法从情感的角度理解世界，自以为拥有了智力就拥有了一切。
> 但是生命需要智力，但生活需要智慧，我们一直在谈“自洽”，就是希望能平衡思想与行为，只有发自内心地理解世界，才能做出善意的行动。

## 生活

**最后的总结，留给生活，今年是重塑自己生活的一年**

### 方寸之间
今年年初搬了一次家，在公司附近的房子其实都不算十分称心，但是看到的这个一居主要因为它足够空，没有别的家具，所以可以给我自己折腾，于是当天下午我就收拾东西从同事家里搬了出来

收拾收拾了桌面，这一年里在这写了代码，看了书/影/番，玩了许多游戏，我买了一台咖啡机，在这喝了很多咖啡，熬了很多很多睡不着的夜晚，这一片方寸之地俨然成为今年生活中重要的一部分

![](/postImg/2024-summary/desktop.jpeg)

### 聚散

*今年没有少聚*

去杭州跟三五好友聚一块吃了饭，一块逛了阿里巴巴园区（对的，就是去别人公司里面逛了一圈），还有一块走了一会西湖

跟大学的师兄一块在上海边喝咖啡边city walk，骑车闲聊

朋友来南京一块逛了秦淮河，水游城（二次元浓度一下拉满了），最后三人写码到了凌晨三点挤在了一张床上😷

还有跟朋友一块坐轮渡，吹着江风吐槽着工作上的事

……

*既有好聚，也有好散。*

跟离职的同事，还有赴京的同事，都好好的吃了一顿饭做道别，祝以后都前程似锦。~~别忘了提携小弟~~

### 学艺
去年的flag里有提到学一门特长，所以就有了今年的拜师学艺🎹

有人学弹琴是为了爱好，有人是深陷艺术，有人是表达自我，

**我比较世俗，我就是想在一个合适的时机装把大的😅**

我的老师跟我在闲聊的时候也说到过这个事情，他说

> 我就是以前看周杰伦弹琴很帅

不是一家人，不进一家门，不然怎么能当我老师呢😝

虽然动机不纯以及工作原因最多周末去一次的缓慢进展，但是努力的过程总有它的意义，生活中也需要那么一些浪漫的瞬间

在巨大的城市机器面前，在我的小小出租屋里，在黑白交错的琴键上，此刻我拥有与生活共舞的能力

![](/postImg/2024-summary/piano.jpeg)

### 独行

> 与独处相安，与万事言和

不排斥三五朋友相伴，但是也不排斥自己一个人的时间，安静，自在，不拘束

自己一个人深夜突发奇想爬去了紫金山看日出（虽然最后也没看到，给云挡住了，可恶啊  

自己一个人去吃想吃的日料，自己一个人到处乱逛，自己一个人看电影，自己一个人泡书店，自己一个人在ktv唱到深夜

这一年里独自一人，那就自己走向自己

# 2025

## Learning, Work

> 不是天才，就要慎独

+ 更加专注在技术上的深度，不要浅尝辄止
+ 既然钢琴学了，那就慢慢学下去
+ 挑战一些新东西，新的技术方向，新的爱好特长……
+ 书还是要好好看的，也是要好好思考的

## 生活

+ 锻炼身体，试着坚持跑步🏃
+ 身体健康
+ 试着多去一些地方
+ 理财，试着节约一些不必要的开销

# 最后

> There is no path to happiness, happiness is the path.

很多事情在今年没有完成，很多问题今年也没有答案，有很多事情努力了，但好像离期望仍然差了很远。

**但是时间、人生都无法暂停，既然往者不可諫，那么就去码头整点薯条吧**

无论任何人，很高兴在这以文字的方式遇见你，希望你一切顺利

祝大家圣诞节快乐🎄，也提前祝大家新年快乐🧨
